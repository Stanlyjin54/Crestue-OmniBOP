### 3. 使用DataMapper进行数据映射

```javascript
import DataMapper from '@/utils/dataMapper.js'

// 单个产品数据映射
const vipmroProduct = {
  id: 'vip001',
  product_title: '工业级螺栓',
  price_info: {
    current_price: 12.5,
    original_price: 15.0
  }
}

const mappedProduct = DataMapper.mapProductData(vipmroProduct, 'vipmro')
console.log('映射后的产品:', mappedProduct)

// 批量数据映射
const products = [vipmroProduct1, vipmroProduct2, gongbangbangProduct]
const mappedProducts = await DataMapper.mapBatchData(products, 'vipmro', (progress) => {
  console.log(`映射进度: ${progress.percentage}% (${progress.current}/${progress.total})`)
})

// 数据验证
const validation = DataMapper.validateProductData(mappedProduct)
if (!validation.valid) {
  console.error('数据验证失败:', validation.errors)
}

// 数据清洗
const cleanData = DataMapper.cleanProductData(mappedProduct)
```

## 🐛 问题排查

### 常见问题

1. **数据映射失败**
   - 检查供应商平台数据格式
   - 验证必填字段是否存在
   - 查看控制台错误信息

2. **价格显示异常**
   - 确认价格数据结构正确
   - 检查货币代码是否支持
   - 验证阶梯价格格式

3. **移动端样式错乱**
   - 检查viewport设置
   - 验证CSS媒体查询
   - 确认组件导入正确
